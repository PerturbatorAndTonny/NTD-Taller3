# 📌 Gestor de Proyectos Web

## 📖 Descripción

Este proyecto consiste en la construcción de un sitio web que permite:

-   Crear proyectos
-   Asignar miembros a cada proyecto
-   Gestionar tareas dentro de cada proyecto
-   Asignar responsables a cada tarea

La aplicación fue desarrollada utilizando tecnologías base del
desarrollo frontend:

-   HTML
-   CSS
-   JavaScript (Vanilla JS)
-   localStorage para persistencia de datos

------------------------------------------------------------------------

## 🚀 Funcionalidades

### ✅ Gestión de Proyectos

-   Crear nuevos proyectos
-   Visualizar proyectos en una grilla dinámica
-   Persistencia de proyectos usando localStorage

### 👥 Gestión de Miembros

-   Asignar miembros a cada proyecto
-   Visualizar miembros asociados

### 📝 Gestión de Tareas

-   Crear tareas dentro de un proyecto
-   Asignar responsables a cada tarea
-   Mostrar tareas organizadas por proyecto

------------------------------------------------------------------------

## 🏗️ Arquitectura Básica

La aplicación sigue una estructura simple basada en:

-   **Estado**: Guardado en `localStorage`
-   **Renderizado dinámico**: Generación de elementos del DOM mediante
    `createElement`
-   **Separación de responsabilidades**:
    -   Funciones para guardar datos
    -   Funciones para obtener datos
    -   Funciones para renderizar la interfaz

## ▶️ Cómo Ejecutar el Proyecto

1.  Descargar o clonar el repositorio.
2.  Abrir el archivo `index.html` en un navegador.
3.  Comenzar a crear proyectos y tareas.

No requiere instalación ni dependencias externas.
